##### float
float은 자기 자신의 위치를 주변의 콘텐츠로부터 상대적으로 배치하는 속성입니다. 다단 컬럼형 레이아웃을 시도할 때 매우 유용하죠.
* float된 엘리먼트가 부모 엘리먼트의 높이에 영향을 주지 않는다는 사실은 경험을 통해 알 수 있음.


- float & clear: both
- float하면 도대체 height랑 margin, padding 이런게 어떻게 되는거냐. 얘는 왜이렇게 막무가내냐.
